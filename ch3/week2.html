Planning Document

Use Case – Create Hello World Program

Scope: Hello World Program as an Applet

Level: User goal

Primary Actor: Person who loads clicks on the link to the Hello World page

Stakeholders and Interests:

-Person who loads the Hello World page

      Is interested to successfully load the applet with image and see the full page “Hello World”

Pre-conditions:

Internet connection is established.

The web page is loaded successfully.

Success guarantee – Post conditions:

The Hello World program is run successfully and viewed on the browser.

Main Success Scenario (or Basic Flow):

1. User clicks on the link directing to the Hello World page.

2. The web page opens.

3. The contents of the web page (including the applet) are displayed.

Extensions or Alternative Flows:

1. At any time, the internet connection is lost.

a. Error message displayed requesting the user to check his internet connection.

2. The user clicks the back button before the program is loaded fully.

b. The program terminates and the previous web page is displayed.

/*Code to implement Hello World program as an Applet*/
import java.io.*;
import java.awt.*;
import java.applet.Applet;
public class HelloWorld_Assign extends Applet {        /*the class HelloWorld_Assign extending the built-in Applet class*/
public void paint (Graphics g)
{   /*Attributes to define the Applet*/
    Dimension d = getSize();                                           
    g.setColor(Color.white);
    g.fillRect(0, 0, d.width, d.height);
    Font f = new Font("Courier-New",Font.BOLD, 18);
    g.setFont(f);
    Color c = new Color(0,150,200);
    g.setColor(c);
    g.drawString("Hello World!!", 40, 40);
    Image i= getImage(getCodeBase(), "hello_world.jpg");
    g.drawImage(i, 20, 60, this);

}
